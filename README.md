# Small-Cox, Big Projects

A small static site showcasing Tyler and Mallory's renovation and maker projects.

- Live site (GitHub Pages): https://smallcox-bigprojects.github.io/BigProject/
- Claude project: https://claude.ai/project/019d1667-bccf-7687-9ab3-11d236474761

Repository contents

- `project-labs.html` — Lil' Projects / 3D prints page

Quick local preview

- Open `index.html` in your browser, or run a simple server:

```bash
# Python 3
python -m http.server 8000
# then open http://localhost:8000/
```
